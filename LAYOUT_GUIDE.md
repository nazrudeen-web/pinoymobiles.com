## RESPONSIVE BREAKPOINTS

### Mobile (< 768px)

- Hero: Stack vertically
- Cards: 1 column
- Categories: 2 columns
- Brands: 2-3 columns
- Menu: Hamburger navigation

### Tablet (768px - 1024px)

- Hero: Partial side-by-side
- Cards: 2 columns
- Categories: 4 columns
- Brands: 5-6 columns

### Desktop (> 1024px)

- Hero: Full side-by-side with image
- Cards: 3 columns
- Categories: 8 columns
- Brands: 10 columns
- Full-width features

## COLOR SCHEME

**Background Colors**:

- Primary BG: `white` (#FFFFFF)
- Section BG: `bg-muted/40` (light gray)
- Dark BG: `slate-900` (footer)

**Accent Colors**:

- Primary: Blue/Purple
- Secondary: Light Purple
- Success: Green (Budget section)
- Alert: Red (Cons section)
- Warning: Orange/Yellow

**Borders**:

- Default: `border-border/40` (light)
- Hover: `border-primary/20`

**Text Colors**:

- Headings: `foreground` (very dark)
- Body: `muted-foreground` (medium gray)
- Links: `primary` (blue)
